# ImproveU: Improve Your Productivity
## The main goal of this app is to make its users more productive by using the Eisenhower principle which was invented by Dwight D. Eisenhower. It will help to prioritize tasks by urgency and importance. Other features are taking short notes and managing personal expenses. It has also an extra feature which is showing daily inspirational quotes from the worldâ€™s famous influential figures to keep the user motivated.

### Main Features:
- Prioritize tasks by urgency and importance
- Take notes
- Manage expenses

### What's New:
- Get a random inspirational quote everytime opening the app

### Supported System: Android 5.0+
### Programming Language: Java
### Database: Firestore
